## Usage

### Development

Just run and visit http://localhost:3000

```bash
pnpm dev
```

OR

```bash
yarn dev
```

### Build

To build the App, run

```bash
pnpm build
```

OR

```bash
yarn build
```

And you will see the generated file in `dist` that is ready to be served.

### Deploy

Deploy anywhere.
